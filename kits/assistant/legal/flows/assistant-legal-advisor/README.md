# Assistant Legal Advisor Flow

This folder is reserved for the Lamatic flow export used by `kits/assistant/legal`.

Replace the placeholder files in this directory with your exported Lamatic flow package:
- `config.json`
- `inputs.json`
- `meta.json`
- `README.md`

Recommended flow behavior:
1. Accept legal question input, optional jurisdiction, and optional context.
2. Retrieve relevant statutes/case law/documents.
3. Return an informational legal summary.
4. Include references.
5. Include recommended next steps.
6. Include a clear disclaimer that output is not legal advice.
